10.1.3 特别针对一个发布风格的字符串

Markup strings 是一个特定风格标记文本使用的字符串。

这些字符串涵盖最基本的标记类，它们在各种风格之间的操作差别很小。

可用的标记字符串

image-with-desc
一个图片和一个描述。

参数 1：不带扩展名的图片；参数 2：图片扩展名；参数 3：描述。

image
一个直接插入的图片。

参数 1：不带扩展名的图片；参数 2：图片扩展名。

image-link
到一个图片的一个链接。

参数 1：链接；参数 2：不带扩展名的图片；参数 3：图片扩展名。

anchor-ref
到当前页面一个锚的引用。

参数 1：锚名；参数 2：如果存在就是描述，否则是原始的链接。

url
不带描述的一个 URL。

参数 1：URL。

link
带描述的一个链接。

参数 1：链接；参数 2：如果存在就是描述，否则是原始的链接。

link-and-anchor
到含有锚的一个页面链接和一个描述。

参数 1：链接；参数 2：锚名；参数 3：如果存在就是描述，否则是原始的链接。

email-addr
到一个 email 地址的一个链接。

参数 1：email 地址；参数 2：email 地址。

emdash
两个 - 长的破折号。

参数 1：起始的空白字元；参数 2：终止的空白字元。

comment-begin
注释的开始。
comment-end
注释的结束。
rule
一个水平线或者空白。
no-break-space
一个分离两个不会被分开的单词的间隔。
footnote
脚注的开始。
footnote-end
脚注的结束。
footnotemark
为当前脚注标记一个引用。

参数 1：该脚注的数字标号。

footnotemark-end
当前脚注引用的结束。
footnotetext
显示当前脚注的文本。

参数 1：该脚注的数字标号。

footnotetext-end
脚注文本行的结束。
fn-sep
用来替代“Footnotes:”行的文本。
dots
三个点。
enddots
四个点。
part
一个 part 显示行的开始，用于书的发布。
part-end
一个 part 显示行的结束，用于书的发布。
chapter
一个 chapter 显示行的开始，用于书的发布。
chapter-end
一个 chapter 显示行的结束，用于书的发布。
section
第一层 section 显示行的开始。

参数 1：section 层数；总是1。

section-end
第一层 section 显示行的结束。

参数 1：section 层数；总是1.

subsection
第二层 section 显示行的开始。

参数 1：section 层数；总是2。

subsection-end
第二层 section 显示行的结束。

参数 1：section 层数；总是2。

subsubsection
第三层 section 显示行的开始。

参数 1：section 层数；总是3。

subsubsection-end
第三层 section 显示行的结束。

参数 1：section 层数；总是3。

section-other
大于三层的 section 显示行的开始。

参数 1：section 层数。

section-other-end
大于三层的 section 显示行的结束。

参数 1：section 层数。

begin-underline
下划线文本的开始。
end-underline
下划线文本的结束。
begin-literal
抄录文本的开始。它包括 <code> 标签和 =teletype text=。
end-literal
抄录文本的结束。它包括 <code> 标签和 =teletype text=。
begin-emph
第一层强调文本的开始。
end-emph
第一层强调文本的结束。
begin-more-emph
第二层强调文本的开始。
end-more-emph
第二层强调文本的结束。
begin-most-emph
第三层（最后）级强调文本的开始。
end-most-emph
第三层（最后）级强调文本的结束。
begin-verse
诗的开始。
verse-space
用于比诗的开始进一步被缩进的每个空白的字符串。
begin-verse-line
诗的一行的开始。
empty-verse-line
诗的一行的结束。
begin-last-stanza-line
一个诗章最后行的开始。
end-last-stanza-line
一个诗章最后行的结束。
end-verse
诗的结束。
begin-example
一个实例区域的开始。要使用这个字符串，一个 `<example>' 标签是必须的。
end-example
一个实例区域的结束。要使用这个字符串，一个 `</example>' 标签是必须的。
begin-center
开始一个居中的行。
end-center
结束一个居中的行。
begin-quote
开始一个引用的区域。
end-quote
结束一个引用的区域。
begin-uli
开始一个无序列表。
end-uli
结束一个无序列表。
begin-uli-item
开始一个无序列表项。
end-uli-item
结束一个无序列表项。
begin-oli
开始一个有序列表。
end-oli
结束一个有序列表。
begin-oli-item
开始一个有序列表项。
end-oli-item
结束一个有序列表项。
begin-dl
开始一个定义列表。
end-dl
结束一个定义列表。
begin-dl-item
开始一个定义列表项。
end-dl-item
结束一个定义列表项。
begin-ddt
开始一个定义列表的术语。
end-ddt
结束一个定义列表的术语。
begin-dde
开始一个定义列表条目。
end-dde
结束一个定义列表条目。
begin-table
开始一个表格。
end-table
结束一个表格。
begin-table-group
开始一个表格组。
end-table-group
结束一个表格组。
begin-table-row
开始一个表格行。
end-table-row
结束一个表格行。
begin-table-entry
开始一个表格条目。
end-table-entry
结束一个表格条目。
